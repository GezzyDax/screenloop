# Code of Conduct

Screenloop follows the Contributor Covenant Code of Conduct.

## Our Pledge

We aim to keep the project professional, respectful, and useful for people running Screenloop in homes, offices, clinics, shops, and other local-network environments.

Participants are expected to:

- Be respectful and constructive.
- Focus criticism on technical work, not people.
- Avoid harassment, personal attacks, or discriminatory language.
- Report security issues privately instead of publishing exploit details.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that are abusive, unsafe, spammy, or unrelated to the project.

Serious or repeated violations may lead to a temporary or permanent ban from project spaces.

## Reporting

For behavior concerns, contact the maintainers through GitHub. For vulnerabilities, use GitHub Security Advisories when available.
